# **Taller: Implementación de la clase `RISEncoder` (Formato RIS)**

**Curso:** Programación Orientada a Objetos
**Tema:** Interfaces, clases genéricas, codificadores y manejo básico de archivos.

## **1. Objetivo del taller**

Implementar un codificador en Java que convierta objetos de tipo `Reference` al **formato RIS**, utilizando la interfaz `Encodable<T>`.
El estudiante debe **investigar por su cuenta** la estructura del formato RIS y construir las clases necesarias.

## **2. Interfaz proporcionada (obligatoria)**

```java
import java.util.List;

public interface Encodable<T> {

    public abstract String[] getFieldNames();
    public abstract String[] getFieldValues(T type);
    public abstract String encode(T type);
    public abstract String encode(List<T> list);
}
```

## **3. Actividades**

### **Actividad 1 — Investigar el formato RIS**

 Debe investigar:

* ¿Qué es el formato RIS?
* ¿Qué etiquetas incluye (`TY`, `AU`, `TI`, `PY`, `ER`, etc.)?
* ¿Qué etiquetas son obligatorias?
* ¿Cómo se agregan múltiples autores?
* ¿Cómo debe terminar un registro RIS?
* Ejemplos de archivos `.ris`.

### **Actividad 2 — Crear la clase `Reference`**

Crear una clase:

```java
public class Reference { ... }
```

con **todos los nombres de atributos y métodos escritos en inglés**.

Debe incluir como mínimo:

* `String title`
* `List<String> authors`
* `String venue` (journal, conference, etc.)
* `String doi`
* `String abstractText`
* `int year`

### **Actividad 3 — Implementar la clase `RISEncoder`**

Crear:

```java
public class RISEncoder implements Encodable<Reference> { ... }
```

* Implementar todos los métodos definidos en `Encodable<T>`.
* Elegir qué etiquetas RIS usar y en qué orden.
* Generar correctamente múltiples líneas `AU -` si hay varios autores.
* Asegurarse de que cada registro termine con la etiqueta `ER`.
* Generar un texto RIS válido.

### **Actividad 4 — Crear la clase `Main` de prueba**

La clase `Main` debe:

1. Crear una lista con **10 referencias** (ver sección de referencias sugeridas).
2. Crear una instancia de `RISEncoder`.
3. Generar el texto RIS correspondiente.
4. Mostrarlo por consola.
5. Guardar la salida en un archivo `.ris` usando el método utilitario proporcionado (Actividad 5).

### **Actividad 5 — Agregar el método para guardar el archivo `.ris`**

```java
import java.io.FileWriter;
import java.io.IOException;

public class FileUtils {

    public static void saveToFile(String fileName, String content) {
        try (FileWriter writer = new FileWriter(fileName)) {
            writer.write(content);
            System.out.println("File saved successfully: " + fileName);
        } catch (IOException e) {
            System.out.println("Error saving file: " + e.getMessage());
        }
    }
}
```

Luego, en `Main`, deben llamar:

```java
FileUtils.saveToFile("output_references.ris", risContent);
```
